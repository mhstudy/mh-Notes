## 1.3 Flink vs SparkStreaming

  ![](https://nightlies.apache.org/flink/flink-docs-release-1.18/fig/bounded-unbounded.png)

|              | **Flink**          | **Streaming**                        |
| :----------: | ------------------ | ------------------------------------ |
| **计算模型** | 流计算             | 微批处理                             |
| **时间语义** | 事件时间、处理时间 | 处理时间                             |
|   **窗口**   | 多、灵活           | 少、不灵活（窗口必须是批次的整数倍） |
|   **状态**   | 有                 | 没有                                 |
| **流式SQL**  | 有                 | 没有                                 |
